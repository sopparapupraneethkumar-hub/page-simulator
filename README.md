# Page Simulator

This project is a simple web-based page simulator built using HTML, CSS, and JavaScript. It is designed to demonstrate how different pages or sections behave when a user interacts with buttons, tabs, or navigation controls. This project is useful for understanding UI behaviour, dynamic content loading, and page switching effects.

## Features
- Simulates multiple pages or sections inside a single web interface
- Smooth switching between pages
- Clean and simple layout
- Fully front-end based (no backend required)
- Easy to customize and extend

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
- `index.html` – Main page of the simulator
- `style.css` – UI design and styling
- `script.js` – Logic for switching views/pages

## How It Works
1. User clicks on a button or menu item.
2. JavaScript switches the visible section.
3. Only one page/section is displayed at a time.
4. The interface updates instantly without refreshing the browser.
